class altar.models.seismic.Moment.Moment¶

Bases: altar.distributions.Uniform.Uniform

The probability distribution for displacements (D) conforming to a given Moment magnitude scale Mw = (log M0 - 9.1)/1.5 (Hiroo Kanamori) M0 = Mu A D It inherits uniform distribution for verification and density calculations, while generates samples for a combined gaussian and dirichlet distributions
